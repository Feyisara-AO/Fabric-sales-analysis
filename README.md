# Fabric Sales Analysis

## Project Overview  
This project focuses on cleaning, analyzing, and visualizing a fabric sales dataset using Excel. The dataset contained missing values and inconsistencies, which were handled to prepare it for analysis.

---

## Tools Used  
- Microsoft Excel  
- Pivot Tables  
- Data Cleaning  
- Dashboard Creation  

---

## Process  
- Cleaned the dataset by handling missing values  
- Analyzed profit and loss across products  
- Identified underpriced and overpriced items  
- Summarized data using Pivot Tables  
- Built a dashboard to present insights  

---

## Key Insights  
- Some products were underpriced, reducing profit  
- Some were overpriced, affecting potential sales  
- Missing data impacted overall analysis accuracy  

---

## Outcome  
Converted raw data into a structured dataset and built a dashboard that clearly communicates business insights for better decision-making.
